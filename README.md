# SmartHome
智能家居diy解决方案，包含以下四点工作
## raspberry的配置
  * domoticz主体安装
  * homebridge安装
  * mqtt服务的配置
## esp8266子设备开发
  * gpio基础控制
    *继电器软件驱动
    *user_key 驱动
  * wifi上网控制
  * mqtt服务数据解析
## 硬件PCB工程
  * 220v-3.3v 稳压供电
  * gpio - 继电器模块驱动
       *- user_key1 上电进入工作模式
       *- user_key2 自定义模式
  * 220v 电源控制接口
  * uart 预留调试接口
  
## siri - domoticz - esp8266 控制
  * 配置手机
  * 数据联调
  * 完成基础控制工作
